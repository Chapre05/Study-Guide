WST151 Study Guide

Key Terms

Accessibility

The inclusive practice of ensuring there are no barriers that prevent interaction with, or access to, websites on the World Wide Web by people with physical disabilities, situational disabilities, and socio-economic restrictions on bandwidth and speed.

Client

The client, or user, side of the Web. It typically refers to the Web browser in the user's machine.

Domain Name

A Domain Name is a unique label for an entity of the internet.

DNS

The Domain Name System organizes the internet by identifying the address and type of entity. It also converts domain names into IP addresses that can be understood by machines comprising the Internet.

FTP

The File Transfer Protocol is a set of rules that for exchanging files over the internet. Unlike HTTP which is used to ask for the entire webpage and it's files to display them, FTP is used only to move a file from one computer to another. FTP is commonly used by web developers when they want to tranfer their local files from their computer to the server and vice versa.

HTML

Hypertext Markup Language is the language used for creating web pages and web applications. HTML has a specified tag set for semantically describing the structure of a web page. HTML elements are the building blocks of a web page and are defined by tags written using angle brackets.

HTTP

Web browsers and web servers communicate using hyperttext transfer protocol. HTTP has rules for exchanging different types of files like text, images, audio, and video. When a browser is directed to a hyperlink, an HTTP request is built and sent to the server of that URL which then processes the request and responds by sending the requested files.

Internet

The internet is an interconnected network of computer networks that spans the globe. Today it is a major part of our lives.

IETF

The Internet Engineering Task Force (IETF) is the principal body engaged in the development of new Internet protocol standard specifications.

IMAP

The Internet Message Access protocol is an Internet standard protocol used by email clients to retrieve email messages from a mail server over a TCP/IP connection.

IP

IP works with TCP and adds additional information to each piece of data such as the IP version and address so that the data can be routed to the correct network. When the IP information is added, the data is often referred to as a packet.

IP address

An IP address is a numeric ID given to each device connected to the internet. Every network has a public IP address and every device in the network has a private IP address. IP addresses sometimes correspond to domain names.

IPv4

Internet Protocol Version 4 (IPv4) is the fourth revision of the Internet Protocol and a widely used protocol in data communication over different kinds of networks.

IPv6

IPv6 is an Internet Layer protocol for packet-switched internetworking and provides end-to-end datagram transmission across multiple IP networks, closely adhering to the design principles developed in the previous version of the protocol, Internet Protocol Version 4 (IPv4).

LAN

A local area network (LAN) is a computer network that interconnects computers within a limited area such as a residence, school, laboratory, university campus or office building.

Markup-Language

In computer text processing, a markup language is the system for annotating a document in a way that is syntactically distinguishable from the text, meaning when the document is processed for display, the markup language is not shown, and is only used to format the text.

MIME

Multipurpose Internet Mail Extensions,MIME is a specification for the format of non-text e-mail attachments that allows the attachment to be sent over the Internet

Network

Computer network, two or more computers that are connected with one another for the purpose of communicating data electronically.

Packets

A packet is the unit of data that is routed between an origin and a destination on the Internet or any other packet-switched network.

POP

Post Office Protocol (POP) is a type of computer networking and Internet standard protocol that extracts and retrieves email from a remote mail server for access by the host machine.

Protocols

Protocols are rules that describe how clients and servers communicate with each other over a network. There are many different protocols with different purposes. These protocols are implemented by operating systems, browsers, and networking devices.

RFC

A Request for Comments (RFC) is a formal document from the Internet Engineering Task Force ( IETF ) that is the result of committee drafting and subsequent review by interested parties. Some RFCs are informational in nature.

Server

A server is a computer designed to process requests and deliver data to another computer over the internet or a local network.

SMPT

SMTP (Simple Mail Transfer Protocol) is a TCP/IP protocol used in sending and receiving e-mail.

TCP

TCP (Transmission Control Protocol) is a standard that defines how to establish and maintain a network conversation via which application programs can exchange data.

TCP/IP

TCP/IP, or the Transmission Control Protocol/Internet Protocol, is a suite of communication protocols used to interconnect network devices on the internet. TCP/IP can also be used as a communications protocol in a private network (an intranet or an extranet).

TLD

Top-level domain (TLD) refers to the last segment of a domain name, or the part that follows immediately after the "dot" symbol. For example, in the internet address: https://www.google.com, the “.com” portion is the TLD.

URL

Uniform Resource Locator: a protocol for specifying addresses on the internet. an address that identifies a particular file on the internet, usually consisting of the protocol, as http, followed by the domain name.

W3

The World Wide Web Consortium (W3C)Takes a proactive role in developing recommendations and prototype technologies related to the Web. These recommendations are not rules but guidelines.

XML

XML is a human and machine readable markup language for storing and transporting data. It looks very similar to HTML, having opening and closing tags with attributes, but the tags are user defined.

Questions

What events and technologies helped the internet grow in the early 90s?

Restrictions were lifted in 1991 and the Internet grew. Graphical web browsers and operating systems, easy and cheap internet access.

What is the purpose of organizations like W3C and IETF?

The Internet Engineering Task Force (IETF) is the principal body engaged in the development of new Internet protocol standard specifications. The World Wide Web Consortium (W3C) Takes a proactive role in developing recommendations and prototype technologies related to the Web.

Why should you put extra effort into making websites more accessible?

Providing access for visitors with visual, auditory, and other challenges should be an integral part of web design rather than an afterthought. It's the right thing to do and in some cases it's the law.

What are 3 ways to make websites more accessible?

Use the alt attribute to describe images, use descriptive links, provide a text navigation at the bottom of the page, and use semantic tags like h1, heading, footer.

What is intellectual property?

Intellectual property (IP) refers to creations of the mind, such as inventions; literary and artistic works; designs; and symbols, names and images used in commerce. IP is protected in law by, for example, patents, copyright and trademarks.

When is it ok to use images,audio, or video,created by someone else?

As long as you ask permission or you have permission to use the images, audios and videos for copyright reason
